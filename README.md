# HR-Analytics-Attrition
HR Analytics project using SQL and Power BI to analyze employee attrition

## 📊 Project Overview
Analyzed employee attrition using SQL and built an interactive dashboard in Power BI.

## 🛠 Tools Used
- SQL (PostgreSQL)
- Power BI

## 📌 Key Insights
- High attrition observed among low salary employees
- Young employees have highest attrition rate
- Certain departments show higher turnover
- Most employees leave within early tenure
